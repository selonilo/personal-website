Fisherman Web

Bu web uygulaması balıkçıların balık tuttuğu bölgeleri harita üzerinden seçerek kaydetmesini sağlamak için tasarlanmıştır. Kullanıcı harita üzerinden kaydettiği pinler üzerinden gönderi paylaşıp farklı kullanıcılar ile etkileşime girebilmektedir.

Demo Uygulama

Fisherman Mobil

Fisherman Web uygulamasının mobil versiyonudur. Kullanıcılar, mobil cihazları üzerinden kolayca balık tuttukları bölgeleri işaretleyip, gönderi paylaşarak diğer balıkçılarla etkileşimde bulunabilirler.
